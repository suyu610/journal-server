## 项目简介

`好享记账`是一个基于flutter开发的，跨端AI记账APP。实现了多人记账、语音记账、截屏快捷记账、自然语言记账等功能。
整体功能完整，前后端全开源。

## 技术栈
前端使用flutter、后端使用java、mysql、redis
